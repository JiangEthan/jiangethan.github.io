如何升级订阅开通 Claude Pro - Claude 3.5 账号注册教程（2024年更新）

发表于2024-12-02|更新于2024-12-02

Claude Pro 是什么？

Claude Pro 是对原版 Claude 3.5的升级，升级后用户能优先体验更先进的AI模型和增强的功能，如：交互面板、文件上传处理、调整写作风格、强大的编程辅助等。和Poe、Cursor、ChatGPT等 AI 工具一样，Claude Pro也采用会员订阅制的方式升级。

Claude 账号如何注册？

很多小伙伴claude账号注册被封的原因是：频繁更换 IP 节点，使用被污染的节点注册登录（常见于一些免费的节点），在Claude 不支持的地区注册登录（例如：香港台湾的节点）和注册需要的手机号归属地在不支持的地区。

注册Claude账号需要具备以下条件：

  1. 有支持Claude 条款地区的上网环境
  2. 谷歌账号（参考谷歌注册教程, 2024.12.02亲测该方法依然有效）

这里建议大家，更改语言后，谷歌浏览器开启无痕模式，再走上面的注册教程，成功率会更高！

注意：注册用到的手机号验证，可以使用虚拟卡平台提供的，下面步骤会提到。如果是mac系统更改chrome语言不成功，可以试试更改系统地区和语言

具体步骤如下：

  1. 打开Claude AI 官网

  2. 这里要用 Claude 条款支持地区的手机号，我用的是一个虚拟卡平台提供的专门注册Claude 的手机号，成功率非常高，几乎不封号！

  3. 申请前需要先在这个虚拟卡平台注册哦，注册后还可以在这个平台订阅Claude和ChatGPT Plus，点击【立即开通】就行，另外一个是镜像服务（镜像其实就是账号共享），注意哦~要先注册才能申请手机号哦（这里建议大家还是自己注册一个Claude账号，镜像的话因为多人共享使用，会存在数据污染导致回答质量不佳的情况）

  4. 选中Claude，然后点【确定】，会提供一个手机号给你，注意在号码有效期内完成验证哈

  5. 接着到Claude注册页面，选择和号码相同的地区，然后粘贴刚刚的手机号，发送验证码

  6. 回到WildCard页面查看验证码，把验证码回填就注册好了

以上用到的虚拟卡平台是：WildCard

Claude Pro 会持续更新

Claude Pro 是 Anthropic 推出的付费版本，提供了最新的功能和改进，会持续更新以确保用户获得最佳的使用体验。

Claude Pro 和原版 Claude 3.5 Haiku 的 4 项差异

1. 体验最新 AI 模型 Claude 3.5 Sonnet、 Claude 3.5 Opus、MCP

升级至 Claude Pro，用户可优先使用最新的 AI 模型，包括 Sonnet 、 Opus和Haiku，更新版本的模型性能和处理能力相比普通用户使用的基础模型更为强大，能更智能地处理复杂的对话和任务。

Anthropic 在 2024 年 11月推出了最新的 Model Context Protocol（MCP，翻译过来就是模型上下文协议），可以让开发人员构建AI应用。

2. 图片生成功能

除了文本生成，Claude Pro 还支持图像生成功能。用户可以输入描述性文字，然后 AI 会生成相应的图像，适用于创意设计、内容创作等多种场景，提升用户在视觉创作上的可能性。

3. 建立与使用 AI 机器人搭建工作流（MCP）

该功能非常适合开发人员使用，配合Claude 3.5 Sonnet 快速构建 MCP 服务器。

搭建流程如下：

  1. 下载Claude 桌面应用程序
  2. 查看官方快速入门文档
  3. 开发过程搭配MCP规范和SDKs 和 MCP 服务器开源存储库 食用

升级 Claude Pro 实际使用经验分享 (2024 年更新)

我是从今年 7 月份就一直订阅 Claude Pro，相比于 GPT-4 ，个人认为 Claude Pro 有以下优点。

回答算法编程等复杂问题表现出色

我测试了几次和编程有关的复杂问题，发现其在算法设计、优化建议以及数据分析方面，生成的答案逻辑清晰，还能提供详细的步骤和代码示例，解题能力明显优于其他模型。

多种写作风格可供选择/非常适合内容创作

新版写作风格的选择 和 界面的流畅性给用户交互的即时反馈感很好。这个功能非常适合内容创作者~

或许你不需要 Claude Pro？

对于日常对话需求，原版 Claude 可能已经足够了（PS：目前免费账号使用的模型是 Claude 3.5 Haiku，且每天有100次额度）。如果对高级 AI 功能需求较低，可不必考虑升级。

如果你已经有 Claude 账号，但是没有升级到 Pro，想体验一下 Claue 最新功能的话，可以加v：hylocomod 备注”体验号”，觉得新功能还可以再考虑升级 Pro 哈

目前市场上除了 Claude 外，还有其他比较受欢迎的聊天机器人，如 ChatGPT 和 Gemini 。每个聊天机器人都有其优缺点，建议多方尝试。

不过需要注意的是：目前 Claude 对账号封控非常严格，大家最好用Google账号注册，且Google账号的地区和你科学上网的地区最好保持一致

需要升级 Claude Pro 吗？

判断是否需要 Claude Pro 可以基于工作需求和对功能的依赖程度。
如果你是 Claude Pro 的重度使用者，每天需要使用几个小时以上，并且希望获得更快的速度、更好的体验和最新的功能，那么我认为升级到 Claude Pro, 可能性价比会更高。

如果你平常使用的是其他的AI工具,想体验一下目前 Claude Pro 的最新功能,那么可以试试我之前用过的镜像站

了解更多: Claude 镜像站

如果你只是偶尔使用 Claude Pro，或者主要用于聊天等休闲目的，那么原版 Claude Pro 就能满足你的需求。

如何升级 Claude Pro？

注意：市面上存在许多 Claude Pro 共享账号或代充值服务，但这些方式往往不稳定，或者存在反应迟缓、限制较多的问题。因此，自己申请一个专属的 Claude Pro 账号，使用体验绝对是最好的。

Claude Pro 的费用是每月 20 美元 (约 150 米)，如果你想要升级，又没有Claude条款支持地区的信用卡，那么可以参考以下步骤：

  1. 我用的是 wildcard 虚拟卡 这个平台, 用这个最开始是图方便，支持支付宝，不过也有确缺点：注册 WildCard 需要 10.99 美元（约70米），大家填写我的邀请码：GPT112 可以减免1美元开卡费

  2. 选择 Claude，然后【下一步】，当然这个平台也支持订阅其他的付费服务哦

  3. 选择余额支付或线上支付，支付的时候会加收0.7%的手续费，这是因为外汇程序比较复杂，手续费算下来大概1块钱这样（建议确定自己要用的时候再充哈）

  4. 跳转到 Claude 网页，登录账号后点击【Upgrade】

  5. 接着会出现一个订阅的弹窗，弹窗内容描述了升级后的差异，这时你需要点选 【Subscribe to Pro】 进到付款页面

  1. 把虚拟信用卡的信息，复制填到相应位置即可

  2. 最后点击【subscribe】就可以啦！
