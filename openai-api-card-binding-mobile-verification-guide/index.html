OpenAI绑卡和Key申请教程: 如何调用OpenAI的API接口？如何通过手机号验证？

想要调用OpenAI的API接口，但不知道如何开始？本教程详细介绍了如何绑定银行卡、申请API Key以及通过手机号验证，为你提供从申请到验证的一站式解决方案。

0. 注意事项

在你把银行卡绑定到OpenAI之前，务必确保卡里至少有5美元余额。没有这个金额，绑定就会失败哦。还有，一定要通过美国家庭网络环境进行操作，用其他普通的代理网络可能也会导致绑卡过程出问题。

1. 安装浏览器插件

点击查看WildCard注册教程, 填写邀请码：GPT112，这样你能立减2美元哦~

2. 登录 OpenAI platform

先开启你的浏览器，输入platform.openai.com这个网址。接下来，输入你的OpenAI账号和密码，这就是你的ChatGPT账号。

3. 绑定付款方式

登录后，往左边看，有个“Settings -> billing”的选项，点一下，你就能找到充值的地方了，选择“Add payment details”来添加你的支付信息。

如果你想直接跳到这一步，可以用这个链接：https://platform.openai.com/account/billing/overview

接着，选择“Set up paid account”，根据你的需求选是个人使用还是公司使用。

然后，你得输入卡号、有效期、CVV和邮编，下面还要写上你的名字和WildCard提供的账单地址，完成这些后点击“continue”。

绑卡成功后，系统会让你选充值金额，可以是5到50美元之间的任意金额。充值完毕后，你就可以直接使用GPT-4的API了。

此外，你还可以设定自动充值。比如说，如果余额低于5美元，系统就会自动从你绑定的卡里扣10美元来充值。

4. 申请 API Key

点开菜单里的“API Keys”，你就能看到已经创建的API密钥和添加新密钥的按钮。或者，你也可以直接通过这个链接创建新的API Keys：https://platform.openai.com/api-keys

接下来，点击“Start verification”开始验证过程，这时候需要一个海外手机号码。

去WildCard网站，找到“海外手机号”服务，点击“申请手机号”。申请成功后，把得到的手机号复制下来。

在选择国家的部分，记得选英国（United Kingdom），然后把手机号粘贴到OpenAI注册页面的输入框里，点击“发送验证码”。

回到WildCard页面，按一下刷新按钮。这时，你应该能看到一个6位数的验证码显示在屏幕上。复制这个验证码，然后回到OpenAI的页面上填写，这样就能完成注册了。

如果验证码收不到，或者提示手机号已经被注册了，就再申请一个新手机号。通常来说，试3个号码，至少会有一个能稳定收到验证码。

验证成功后，会有提示告诉你。值得注意的是，每个海外手机号可以用来验证两个API。如果你用同一个手机号再次验证，可能会提示这个手机号已经绑定过一个API了，但这不影响使用，所以不用担心。

然后，就可以正常创建和复制API密钥了。

5. API 消费额度和扣费

OpenAI 会根据你充值的金额来设定你的消费上限。举个例子，如果你充了5美元，那么在使用API的时候，你也就最多能消费5美元。

如果你的卡连续两次扣款失败了，比如说因为卡里余额不够，OpenAI 会锁定这个开发者账号和相关的卡号。

作为开发者，你可以随时在这里查看你的API调用消费情况。同时，在这里可以查看你的调用额度限制，以及你可以使用的模型，比如GPT-3.5或者GPT-4。
