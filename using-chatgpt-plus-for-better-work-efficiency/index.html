查找论文

当我的研究涉及到我不熟悉的领域时，用ChatGPT Plus的Consensus插件

我的Prompt为：找出计算机图形学中有关渲染三维曲面的相关论文

ChatGPT会根据你提出的要求，精确地查找有关渲染三维曲面的论文

阅读助手

点击右上角 [ Create ] 上传 PDF电子书，然后在GPT-4界面自定义一个GPTs，让其成为我的读书助手，甚至让它作为老师，出题考我

可视化理解工具

当我难以理解某个概念时，使用Whimsical Diagrams插件，将文本内容可视化，制成思维导图，方便理解记忆

比如，我输入Prompt：请将计算机网络osi的7层结构制成思维导图

它生成会外链，点击外链就能看到思维导图了，并且可以将图保存下来

自学助手

如果你对某个领域感兴趣，ChatGPT能够很好的帮你入门这个领域，并且根据你当前的水平为你量身定制学习路线

比如，我问它：我想制作一个水火箭，但我的物理知识储备只有高中水平，告诉我应该学习哪些知识以及如何制作水火箭，最好根据我的知识水平，来制定我的学习路线

配图助手

当我写完一款软件，苦于没有好看的图标时，Logo Creator这个插件正好解决了我的苦恼

我输入：我的软件功能大致是：通过深度相机拍摄的数据，实时计算拍摄物体的体积。请以黑色为主色调为我的软件设计一个图标

如果你对生成的图标不满意，可以继续跟它提怎么修改，哪个地方需要精修？

答疑助教

前面介绍了ChatGPT为我规划了学习路线，当我遇到问题时，可以随时问它

比如，我问它：如何理解空气动力学中空气散度的物理意义

然后对于它回答中有不懂的地方，还可以继续追问下去，让它用更通俗的概念向我解释

比如，问它：怎么理解伯努利原理

扮演历史人物

ChatGPT还可以作为一位历史人物，与你攀谈。我让它扮演中国古代先贤孔子，用论语中的语句与我对话

它会给出引用来源，并加以解释
