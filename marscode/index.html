豆包MarsCode是什么？如何安装使用？注册成功最高可领取16元红包？

发表于2024-10-20|更新于2024-10-27

最近，字节有个活动叫「豆包 MarsCode 合伙人计划」（其实是薅羊毛活动）

活动是这样的，拉一位新用户，双方各领 10 火星币，火星币可按 1:1 比例兑换京东卡，又或者以最高 1:1.6 比例兑换现金。

活动奖励很nice，推广条件很简单，不用实名认证，只要有手机号就行

下面我给大家演示一下（全部操作过程不到2分钟）：

1. 通过邀请链接注册，才有火星币哦

打开邀请链接->：豆包 MarsCode - 合伙人计划 ，用手机号注册加入计划

2.下载VS Code

如果你电脑已经安装有VS Code，或者 JetBrains 公司的软件，这一步可以忽略不看，直接在左边目录索引跳转到 第 3 步

在电脑端，下载 VS Code ，按照下面依次点击安装，按照以下步骤安装使用

3. 安装 MarCode 插件

安装成功后，点击进入左侧【扩展插件】，搜索 MarCode，搜索结果第一个就是（ps：如果搜 MarCode ，不出结果，可以搜 MarC，等待搜索结果即可），接着点击安装插件

s

4. 授权登录 MarsCode-watermark

安装成功后，点击左侧栏的【MarsCode-watermark】，接着点击登录，之后会自动跳转浏览器进行授权，重新回到 VSCode 的时候，就已经完成登录了

5. 完成一次和编程有关的对话

接着和 AI 来一次有效交互，需要等它回答结束（例如：用C++写一个快速排序），奖励就到手了（回到网页刷新，会发现有 10 火星币了，11月1号11点后开始兑换，记得在手机日历上标记一下哦）

END

其实豆包 MarsCode-watermark的功能也挺好的，写代码过程中遇到不懂问他，不会写的也问他，再也不用在其他 AI 工具上提问复制粘贴代码了，直接提升写Code效率好吧

当然啦，如果你身边还有不少学计算机的小伙伴，可以返回网页，【申请加入合伙人计划】，拿自己的专属邀请链接，邀请他们注册使用，安利编程工具的同时，顺便请他们喝了奶茶，何乐而不为呢~

但需要注意哦，不能在同一台设备上的插件登录多个新账号去对话哦，容易被系统风控（毕竟奖励力度这么大，基本的防薅手段还是有的），所以企图只让身边人收短信，在自己电脑上操作，独乐乐薅羊毛的路是行不通的（你如果电脑实在很多，当我没说），建议大家还是正儿八经推荐身边人来注册使用，不要动太多歪心思

现在点击下方图片【识别二维码】即可加入 MarCode 大队
