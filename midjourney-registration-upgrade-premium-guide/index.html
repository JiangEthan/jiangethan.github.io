如果你在自己的网络环境使用订阅 Midjourney 出现各种报错无法成功，那么可以尝试使用 WildCard 提供的远程环境进行订阅，如果你没有WildCard账户，这里点击查看注册

1. 使用海外代理，安装浏览器插件

具体教程在这里：如何稳定访问服务

WildCard界面点击使用海外家庭网络

2. 登录 Discord

2.1 登录 Discord

在浏览器输入 Discord的 网址 https://discord.com/login

输入用户名密码，完成验证后，Discord 会提示你：在新的位置登录，需要去邮箱验证。

2.2 解决异地登录问题，去邮箱复制链接

如果你在新的位置登录，则需要去邮箱进行验证，Discord会给你注册邮箱发一封邮件，进入邮件验证一次就可以了。

这时候，需要你在自己的浏览器环境（非WildCard浏览器的环境），打开邮箱，找到 Discord 发来的邮件，然后点击「Verify Login」按钮。

这时需要重新输入 Discord 的用户名和密码进行登录，登录成功会直接显示 Discord 的网页界面。

3. 登录 Midjourney 进行订阅

登录 Discord 后，就可以登录 Midjourney 进行订阅了。在浏览器的地址栏输入 https://www.midjourney.com/login/
在弹出的 Discord 页面点击授权，然后登录 Midjourney

点击左边栏的 「Manage Sub」 ，即可选择套餐进行订阅

点击 「Subscribe」，在弹出的绑定支付页面，依次输入卡号、有效月份/年份、CVC（即您的 CVV ）、美国的账单地址及姓名即可成功绑定。
